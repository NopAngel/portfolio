2026.04.13 // Because WSL2 might be the best option

WSL>> (?)

I used Linux for quite a while and tried practically every 'famous' distro out there. Honestly, I don't think using Linux as a daily driver is a bad choice, but here’s the issue: I feel like it might not be the 'best' option. One of the main problems with Linux is compatibility; it's simply not compatible with many apps (even some development tools). That’s why I believe WSL is one of the best alternatives, as you get Linux compatibility right within Windows, allowing you to use a wider range of tools. I feel it’s an 'underrated' option (in my opinion :3)
